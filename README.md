# SQL Week 2 Assignment

## Overview

This repository contains my solutions for the SQL Week 2 Assignment. The project focuses on creating and managing a PostgreSQL database for Greenwood Academy. It demonstrates SQL Data Definition Language (DDL), Data Manipulation Language (DML), filtering, operators, aggregate functions, and conditional logic using CASE WHEN.
